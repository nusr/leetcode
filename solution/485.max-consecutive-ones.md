# [485.Max Consecutive Ones](https://leetcode.com/problems/max-consecutive-ones/)

## 问题

## 思路

扫描一遍数组，遇到 1 临时变量加一，否组重置为 0，同时每次与最大值比较。
时间复杂度为 `O(n)`
空间复杂度为 `O(1)`

## 代码

- `js`

```js
/**
 * @param {number[]} nums
 * @return {number}
 */
var findMaxConsecutiveOnes = function(nums) {
  let max = 0;
  let count = 0;
  for (let i = 0; i < nums.length; i++) {
    if (nums[i]) {
      count++;
    } else {
      count = 0;
    }
    if (count > max) {
      max = count;
    }
  }
  return max;
};
```
